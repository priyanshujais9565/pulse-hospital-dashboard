# pulse-hospital-dashboard
🏥 Pulse Hospital Dashboard (Power BI)
📊 Project Overview

The Pulse Hospital Dashboard is an interactive Power BI project designed to analyze hospital operations, patient data, and overall performance.

This dashboard helps in tracking key healthcare metrics and supports better decision-making through data visualization.

🎯 Objectives
.Monitor hospital performance in real-time
.Analyze patient admissions and discharge trends
.Track revenue and operational efficiency
.Improve decision-making using data insights

📁 File Included
Pulse Hospital DASHBOARD.pbix → Power BI dashboard file

🛠️ Tools & Technologies

.Data Visualization Techniques
